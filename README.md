# User List App

Небольшое React-приложение для управления списком пользователей.

## Функциональность
- добавление пользователя
- редактирование пользователя
- удаление пользователя
- поиск по имени
- валидация формы
- сохранение данных в localStorage

## Технологии
- React
- TypeScript
- Ant Design
- Vite

## Запуск проекта
```bash
npm install
npm run dev
